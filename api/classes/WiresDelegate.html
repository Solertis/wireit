WiresDelegate Class

Defined in: src/wires-delegate/js/wires-delegate.js:5
Module: wires-delegate

WiresDelegate is an extension for Widgets to manipulate a list of wires.

The WidgetParent/WidgetChild relationship isn't sufficient because wires have 2 parents, so we use this extension instead of WidgetParent
